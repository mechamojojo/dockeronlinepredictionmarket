# CDP SDK Example

Exemplo simples de uso do CDP SDK da Coinbase.

## Instalação

```bash
npm install
```

## Como Executar

```bash
npx tsx main.ts
```

## O que o script faz

1. Cria uma conta owner
2. Cria uma smart account
3. Envia uma user operation
4. Aguarda a confirmação da operação

## Variáveis de Ambiente

Crie um arquivo `.env` se necessário (o CDP SDK geralmente não requer configuração adicional para este exemplo).

## Requisitos

- Node.js 18+
- npm ou yarn
